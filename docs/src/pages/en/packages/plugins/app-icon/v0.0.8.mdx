## Plugin App Icon

`@brandingbrand/kernel-plugin-app-icon` is a package that generates all the necessary app icon assets for both iOS and Android.

### Getting Started

<div class="flex w-full my-6">
  <CodeCopy args="yarn add -D @brandingbrand/kernel-plugin-app-icon" />
</div>

Add the above noted package and the plugin is avaialble in the Kernel plugin ecosystem.

### Usage

After adding this plugin as a dev dependency it is necessary to add it to the kernel plugins configuration via the `package.json`.

`package.json`

```
  "kernel": {
    "plugins": [
      ...,
      "@brandingbrand/kernel-plugin-app-icon",
      ...
    ]
  },
```

Your Kernel environment should also be updated with the `@brandingbrand/kernel-plugin-app-icon` required props from the defined interface.

`.kernelrc/env/env.<env>.ts`

```
  kernelPluginAppIcon: {
    kernel: {
      appIconPath: 'assets/app-icon',
    },
  },
```

The assets must follow a strict naming scheme.

<div class="my-4 overflow-scroll">
| platform         | filename                            | 
| :--------        | :----------                         |
| ios              | ios-universal.png                   |
| android legacy   | android-legacy.png                  |
| android adaptive | android-adaptive-foreground.png     |
| verbose adaptive | android-adaptive-background.png     |
</div>
